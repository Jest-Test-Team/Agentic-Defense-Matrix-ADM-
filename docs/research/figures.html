Figure 2 — The asymmetry: per-message defense cost

ADM's per-action detection is an embedding + cosine + O(1) window update — no model inference. A per-message LLM guard (Llama Guard) pays a full 8B forward pass per message. On a log axis the gap is 3–4 orders of magnitude — the asymmetry ratio α that lets a streaming defense win the throughput race under automated adversaries.

ADM values measured (./cmd/latency); Llama Guard is a hosted-inference reference band — run ./cmd/baseline with a Groq key to replace it with your measured number.

Figure 3 — Steady-state overhead: the ≤5% claim (lock-free vs mutex)

The SIEM hot path on ADM's lock-free ring buffer costs 310 ns/event vs 618 ns for a mutex-guarded queue of equal capacity (2× lower, bounded O(capacity) memory, near-zero GC). Derived single-core CPU overhead vs event rate: the lock-free path stays under 5% up to ≈160k events/s, where the mutex baseline breaches it at ≈80k.

Figure 4 — Live δ/κ from the running OCI battle

Full-history CDFs from the deployed /api/latency endpoint (n=21,238 blocked attacks, n=2,387 remediations). δ is bimodal: most attacks are blocked at the boundary in ~63 ms (p50), with a tail near 12 s for the few that reach the throttled hosted LLM. κ here is the fully orchestrated containment (green-team poll → HTTP revoke → Docker container restart), p50 ≈ 10.5 s — four orders above the raw kill primitive measured in the harness (509 µs). The gap is the research point: the mechanism is cheap; the latency lives in orchestration (polling + restart), which event-driven kill would collapse.
